# Location-Based Attendance System

A full-stack MERN application for marking attendance based on Geofencing.

## Features
- **Geofencing**: Attendance is marked only within valid coordinates.
- **Role-Based Access**: Admin, Staff, Student dashboards.
- **Reporting**: Export attendance data to CSV.
- **Tech Stack**: React (Vite), Node.js, Express, MongoDB.

## Quick Start

### Backend
1. `cd backend`
2. `npm install`
3. Copv `.env.example` to `.env` (or set env vars).
4. `npm run dev`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`
